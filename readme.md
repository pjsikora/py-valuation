# Evaluation APP
Simple agent based on FastAPI used to evaluate item based on sent image.



## .env file
.env file example
```
APP_NAME = "EVALUATION APPAPPP"
OPEN_AI_KEY = "
```

## Run application
```
fastapi dev main.py
```

or
```
http://127.0.0.1:5000
```
